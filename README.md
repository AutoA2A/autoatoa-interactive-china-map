# 互动中国地图

> 本项目由 [www.autoa2a.org](https://www.autoa2a.org) 「AI 研究院」**多个 AI 协作自动生成**（共 8 个页面）。在线访问： https://AutoA2A.github.io/autoatoa-interactive-china-map/

## 网站简介
本项目是一个互动中国地图网站，通过直观的可视化与交互设计，生动呈现全国各省市的地理与数据信息。用户可在地图上自由探索，获取丰富的区域详情，体验流畅的动态反馈。本项目从代码编写、视觉设计到文档撰写，全程由多个 AI 协作完成，是人工智能辅助前端开发的创新实践。

## 页面与功能
网站共包含 8 个精心设计的页面，核心功能与结构如下：
- **首页与地图主视觉**：作为核心交互区，提供可缩放、可悬停的全国地图，支持按省份点击交互，高亮显示区域边界。
- **省级详情页（4个）**：按地理大区划分，深度展示各省份的基础信息、经济指标与特色数据，配合动态图表呈现。
- **数据看板页**：汇总全国核心宏观数据，支持多维度筛选与动态排序。
- **交互引导页**：提供操作提示与图例说明，降低用户使用门槛。
- **关于项目页**：介绍网站背景、技术栈与 AI 协作流程。

## 多 AI 协作与验收
本项目的落地得益于多个 AI 模型的深度协同：
- **架构与开发**：由编码型 AI 主导完成 HTML/CSS/JS 基础代码生成与地图 SVG 逻辑处理；
- **视觉与交互**：设计型 AI 负责配色方案、UI 组件样式及动态效果参数的输出；
- **内容与校验**：校验型 AI 负责审查代码规范、地理数据准确性及多端兼容性，并完成最终缺陷修复。
各 AI 产出的模块经统一整合后，由人工进行最终验收，确保交互流畅、数据无误且视觉风格统一。

## GitHub Pages 部署与访问(入口 index.html)
本项目采用纯静态前端技术构建，已通过 GitHub Pages 完成部署。访问入口为仓库根目录下的 `index.html` 文件。用户无需配置本地服务器，直接点击线上链接即可秒级加载体验。项目源码同样托管于 GitHub，欢迎克隆部署至任意静态托管服务。
